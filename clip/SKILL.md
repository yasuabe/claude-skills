# clip

会話の文脈を参照し、直近に登場した `sudo` コマンドを特定して、xclip でクリップボード（X11 クリップボード）にコピーする。

## 手順

1. 会話の文脈から、最後に言及された `sudo` で始まるコマンドを特定する
2. そのコマンドを以下の Bash コマンドでクリップボードにコピーする:

```bash
printf '%s' '<コマンド全体>' | xclip -selection clipboard
```

3. コピーしたコマンドをユーザーに表示して完了を伝える

## 注意

- sudo コマンドが複数ある場合は、最後に登場したものを対象とする
- sudo コマンドが見当たらない場合は、その旨を伝える
- xclip が利用できない場合は、コマンドをテキストで表示してユーザーに手動コピーを促す
